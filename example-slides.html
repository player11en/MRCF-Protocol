VISION

Build a lightweight task management tool that lets small teams coordinate work

Build a lightweight task management tool that lets small teams coordinate work without the overhead of enterprise project software.

Problem: Existing tools are either too complex or too limited. Goal: A focused app that handles the 80% of project coordination needs. Success: Teams complete 20% more tasks per sprint within the first month.

CONTEXT

**Audience:** Small software teams (2–10 people).

Audience: Small software teams (2–10 people). Constraints: Must run as a single-page app; no server-side rendering required. Stack: TypeScript, React, Vite, SQLite (via Tauri for desktop builds). Environment: macOS and Windows primary, Linux community-supported.

STRUCTURE (1/3)

Architecture

Architecture

Single-page application with a local-first data layer backed by SQLite. Syncing is optional and handled via a lightweight REST adapter.

src/
  components/   — UI layer (React)
  store/        — State management (Zustand)
  db/           — SQLite access layer (Tauri plugin)
  api/          — REST adapter (optional sync)
