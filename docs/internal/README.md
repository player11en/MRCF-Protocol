# docs/internal

These are internal and legacy documents from the early development of MRCF-Protocol. They are kept for historical context but are not part of the public specification or documentation.

| File | Description |
|---|---|
| `spec_v1_kdoc_legacy.md` | Original v1 spec written in German under the working name "KDOC" (precursor to MRCF) |
| `ziel_origin_notes.md` | Early German-language notes about the project's goals and 3-layer architecture concept |
| `developTasks.md` | Internal dev backlog for the initial 4-developer team (German, references KDOC) |
| `developer2-backlog.md` | Dev 2 (VS Code extension) task backlog |
| `developer4-backlog.md` | Dev 4 (renderer) task backlog |

For the current public specification, see [../mrcf-spec.md](../mrcf-spec.md).
